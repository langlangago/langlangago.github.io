    Ubuntu新增SWAP和调整SWAP大小

    系统 > Ubuntu ubuntu swap
    发布时间 : 2021-02-02 10:13
    字数:106 阅读 :
    评论:
    1. 一、新增SWAP分区

    一、新增SWAP分区

    1. 查看内存情况：free -m

    2. 创建一个新的swapfile文件
      sudo dd if=/dev/zero of=swapfile bs=1M count=2048,
      创建2G的swap，大小为bs*count

    3. 把生成的文件转换成 Swap 文件 :
      sudo mkswap swapfile

    4. 开启新的swap:
      sudo swapon swap

    5. 激活 Swap 文件:
      sudo swapon swapfile

    6. 查看swap内存:
      free -m

    7. 设置开机启动

      cp /etc/fstab /etc/fstab.bak 
echo '/swapfile none swap sw 0 0' | sudo tee -a /etc/fstab
