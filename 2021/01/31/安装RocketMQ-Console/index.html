    安装RocketMQ-Console

    中间件 > RocketMQ RocketMQ
    发布时间 : 2021-01-31 20:11
    字数:126 阅读 :
    1. 一、下载
    2. 二、编译打包
    3. 三、启动
    4. 四、验证

    一、下载

    wget https://github.com/apache/rocketmq-externals/archive/master.zip
unzip master.zip

    二、编译打包

    打包依赖maven，可以通过各种方式安装：

    yum install maven

    编译打包：

    cd /opt/master
mvn clean package -Dmaven.test.skip=true

    三、启动

    mkdir /usr/local/rocketmq-console/
cp target/rocketmq-console-ng-2.0.0.jar /usr/local/rocketmq-console/
cd /usr/local/rocketmq-console/
nohup java -jar rocketmq-console-ng-2.0.0.jar --rocketmq.config.namesrvAddr=127.0.0.1:9876 &

    启动的时候要指定Namesvr地址。

    四、验证

    http://xx.xx.xx.xx:8080 能访问，有数据即可。
